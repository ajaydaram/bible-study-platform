## Major Future Features

1. **Integrate Confessional Links in Bible UI**
   - Display confessional citations (from Creeds.json) directly below each verse, with clickable links to the relevant confession/catechism section.

2. **Full-Text Confession Search**
   - Add a search feature for all creeds/confessions/catechisms, with results linking to both the text and cited verses.

3. **Interlinear Upgrade**
   - Use STEP Bible’s full interlinear data (not just lexicons) for local Greek/Hebrew parsing, including morphology and syntax trees.

4. **Verse/Section Cross-References**
   - For any verse, show all related cross-references, commentaries, and confessional citations in one panel.

5. **Offline-First Mode**
   - Bundle all essential data (Bible, lexicons, creeds, maps) for true offline use—no API dependency for core study.

6. **User Notes & Highlighting**
   - Allow users to add notes, highlights, and tags to verses, confessions, and catechism questions.

7. **Advanced Pulse Diagnostics**
   - Use confessional data to power “Pulse” theological diagnostics—show which confessions support or challenge a user’s doctrinal answers.

8. **Data Update Scripts**
   - Automate periodic updates from GitHub sources (creeds, Bible versions, lexicons) to keep your data fresh.
